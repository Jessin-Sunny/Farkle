# 🎲 Farkle - A Dice Game
Farkle is a fast-paced dice game where players roll six dice to score points through combinations, risking losing their turn’s score if they roll without scoring.
## 🚀 Features
- 🎮 Two-player mode
- 🎲 Dice roll animation
- 🏆 Score tracking
